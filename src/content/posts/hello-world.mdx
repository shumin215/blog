## 블로그를 시작하며

안녕하세요, **Shumin's World**에 오신 것을 환영합니다.

이 블로그는 제가 개발하면서 배우고 경험한 것들을 기록하는 공간입니다.
코드 한 줄, 문제 해결 과정, 그리고 가끔은 일상적인 생각들도 담을 예정입니다.

## 앞으로 다룰 주제들

- **개발 경험기** — 새로운 기술을 배우면서 겪는 시행착오
- **문제 해결** — 삽질의 기록과 해결 방법
- **도구 & 환경** — 개발 환경 설정, 생산성 도구 소개

## 이 블로그의 기술 스택

이 블로그는 [Astro](https://astro.build) + MDX + Tailwind CSS로 만들어졌으며,
Docker를 통해 셀프 호스팅 환경에서 운영됩니다.

```bash
# 로컬에서 실행하기
npm run dev
```

글이 도움이 되셨으면 좋겠습니다. 🙂
